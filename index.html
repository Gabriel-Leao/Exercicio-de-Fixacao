Tag <p>:
Com esssa tag um parágrafo é criado. O enter não faz uma quebra de linha, para isso é utilizada outra tag.

Tag <br>:
Com essa tag conseguimos quebrar a linha, para que o conteúdo continua na linha abaixo. sua sigla significa break row e ela pode ser utilizada tanto em parágrafos quanto me títulos e etc...

Tag <b>:
Essa tag deixa o texto em negrito.

Tag <i>:
Essa tag deixa o texto em itálico. Vai uma pizza?

Tag <a>:
Essa tag serve de redicionamento para outras páginas, imagens, trechos dentro da página etc...
É preciso passar o caminho através do href"". Caso seja um arquivo dentro do projeto basta passar o caminho da pasta, caso seja um link externo basta colocar a URL.

Tag <h1>:
Essa tag é utilizada em títulos, sendo muito importante para a semântica do HTML.
Essa tag é a primeira hierarquia de títulos depois dela temos outras tags indo do <h2> até <h6>. Cada uma sendo uma hierarquia diferente.

tag <img>:
Essa tag é utilizada para adicionar imagens na página.
Basta passar o caminho da pasta onde está armazenada a imagem ou a url caso esteja em um outro site, dento das aspas no href" ". Também podemos descrever a imagem dentro de aspas no alt"", muito importante para pessass com deficiência visual.

Tag <li>:
Essa tag é utilizada para criar uma lista
Quando colocada dentro de uma outra tag pode ser uma lista ordenada ou não ordenada.

Tag <ol>:
Essa tag é utilizada para fazer uma lista ordenada. Com números na frente dos itens que compõem a lista.

Tag <ul>:
Essa tag é utilizada para criar uma tag não ordenada com blonihas pretas na frente dos itens que compõem a lista.
